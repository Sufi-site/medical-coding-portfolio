# Mohamed Sufyan - Medical Coding Portfolio

A professional portfolio website showcasing expertise in medical coding, HCC coding, and healthcare compliance.

## Features

- Responsive design optimized for all devices
- Modern UI with smooth animations
- Professional showcase of skills and experience
- Contact form for potential employers
- Social media integration

## Technologies Used

- Vite
- Tailwind CSS
- Vanilla JavaScript
- Boxicons

## Local Development

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`
